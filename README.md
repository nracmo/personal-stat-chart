# The Ultimate Stat Chart (RPG Style)

A dynamic, interactive radar chart built with HTML, CSS, and SVG. 

## Features
- **Visuals:** RPG-style spider/radar chart.
- **Interactive:** Real-time graph updates as you input numbers (0-10).
- **Categories:**
  - Physical (Pink)
  - Technical (Orange) - *Custom Engineering/Crafting stats*
  - Personality (Green)
  - Metaphysical (Purple)
  - Social (Blue)
  - Mental (Yellow)
- **Tech Stack:** Pure HTML/CSS/JS (No frameworks required).

## How to Use
1. Open the website.
2. Enter values from 0-10 in the input boxes on the right.
3. Screenshot your chart to share!
